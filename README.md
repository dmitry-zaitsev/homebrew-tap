# dmitry-zaitsev/homebrew-tap

Homebrew tap.

## Formulae

- `istoria` — local log viewer; pipe stdout into a native window. Source: <https://github.com/dmitry-zaitsev/istoria>.

## Install

```sh
brew install dmitry-zaitsev/tap/istoria
```

Formula is auto-bumped on each release by CI in the source repo. Artifacts hosted at <https://github.com/dmitry-zaitsev/istoria-releases>.
